Jupyter Notebook issue

Issue:

During the first time you have run this project, you are getting the following error in Jupyter Notebook: 

Exception: Could not find the require environment variables: TASTY_BYTES_APP_USER, TASTY_BYTES_APP_PASSWORD, TASTY_BYTES_APP_ACCOUNT

Cause:

Your Snowflake account credentials didn't make it to the DDE.

Resolution:

To set the account credentials, you can use our Streamlit app.

In the DDE terminal, run the following command:

dde-admin

This will open a Streamlit app within the editor.

Use the credentials from the catalog and set the credentials in the DDE Cloud Credentials Setting app:

  • Use the Test button to verify that the credentials are correct.
  • Use the Save button to save the credentials.
